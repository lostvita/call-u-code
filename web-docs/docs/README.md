# Mini web docs created with React

- use react hook
- use react-router
- use quill.js
- use socket-io-client to synchronize document.
- use requestIdelCallback to achieved auto saving.
- use node.js serve
- use mongodb 

# Getting Started

### run client
```
cd docs

yarn

yarn start
```

### run serve
```
cd server

yarn

yarn serve
```
